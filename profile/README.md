# One4All

### Bot Discord tout-en-un

[![Discord](https://img.shields.io/badge/Discord-Serveur%20Support-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.gg/TfwGcCjyfp)
[![Website](https://img.shields.io/badge/Website-one4all.dev-000?style=for-the-badge&logo=vercel&logoColor=white)](https://one4all.dev)

**One4All** est un bot Discord multifonction concu pour gerer, proteger et animer vos serveurs.
98 commandes, 16 categories, 3 langues — un seul bot.

</div>

---

## Fonctionnalites

| | Categorie | Description |
|---|---|---|
| :shield: | **Antiraid** | Anti-spam, anti-link, anti-mass ban/kick, anti-bot, anti-webhook |
| :hammer: | **Moderation** | Ban, kick, mute, warn, clear, nuke, derank |
| :musical_note: | **Musique** | YouTube, Spotify, SoundCloud, Deezer, Apple Music |
| :coin: | **Economie** | Coins, shop, inventaire, leaderboard |
| :tada: | **Giveaways** | Systeme de giveaways avec base de donnees |
| :envelope: | **Invitations** | Tracking, detection de fakes, leaderboard |
| :floppy_disk: | **Backup** | Sauvegarde et restauration complete de serveur |
| :bar_chart: | **Logs** | Voice, bans, kicks, roles, channels, messages |
| :globe_with_meridians: | **Multilingue** | Francais, English, Romana |

## Stack technique

```
discord.js v14 · discord-player v7 · Sequelize v6 · MySQL · Node.js 16+
```
